# [BOJ] 01647 도시 분할 계획

### 문제 바로가기

>  https://www.acmicpc.net/problem/1647

### 알고리즘

> 그래프이론. MST

### 문제 풀이 및 주의

> #### MST
>
> N개의 노드를 있는 N-1개의 간선
>
> 여기서 2개의 그룹으로 최솟갑을 가지도록 분리하기 위해서는
>
> 가장 비용이 많이 드는 마지막 N-1번째 선을 분리하면 된다.



> #### 크루스칼 알고리즘
>
> 실수 return 값 틀리지 않도록한다.
>
> FindSet에서 P[x]!=x    - >  return P[x]=FindSet[x];
>
> 호출만하고 끝나면 return 되지 않음!!
>
> p[x] 수정하지 않으면 시간초과!!
>
