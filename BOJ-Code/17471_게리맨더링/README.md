# [BOJ] 17471 게리맨더링

### 문제 바로가기

>  https://www.acmicpc.net/problem/17471

### 알고리즘

> 그래프 이론. 그래프 탐색. 브루트포스 알고리즘

### 문제 풀이 및 주의

> **브루트포스 알고리즘** [brute force]
>
> Brute: 무식한 . force:힘
>
> 완전탐색 알고리즘. 가능한 모든 경우의 수를 모두 탐새갛면서 요구조건에 충족되는 결과만을 가져온다. 이 알고리즘의 강력한점은 예외없이 100% 확률로 정답만을 출력한다.
>
> 모두 다 해보는 알고리즘
